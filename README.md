# LLM From Scratch

This project follows a 7-step roadmap to build a custom AI model.

## Roadmap

- [ ] **STEP 1**: Tokenizer (Build custom vocab)
- [ ] **STEP 2**: Dataset Scraping (Gather text data)
- [ ] **STEP 3**: Dataset Cleaning (Normalize and format)
- [ ] **STEP 4**: Model Architecture (GPT-style Transformer)
- [ ] **STEP 5**: Pretraining (Train on GPU)
- [ ] **STEP 6**: Fine-tuning (Instruction tuning)
- [ ] **STEP 7**: Serving (Inference API)
